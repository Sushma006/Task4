# VLSI Design Internship – Task 4

## RTL Design of FSM and Control Units

### Overview

This project demonstrates the implementation of Finite State Machines (FSMs) using Verilog HDL.

## Designs Implemented

1. Moore FSM
2. Mealy FSM
3. Traffic Light Controller
4. Sequence Detector (1011)

## Tools Used

- Verilog HDL
- Icarus Verilog
- GTKWave
- EDA Playground

## Learning Outcomes

- FSM Design
- Moore and Mealy Machines
- State Transitions
- Pattern Detection
- Traffic Control Logic
- Verilog RTL Coding
- Testbench Development
- Waveform Analysis
